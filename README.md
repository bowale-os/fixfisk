## Getting Started

Clone the repository and install dependencies for both the client and server:

```bash
git clone https://github.com/bowale-os/fixfisk.git
cd fixfisk

# Client setup (in /client)
cd client
npm install
npm start

# Server setup (in /server)
cd ../server
npm install
npm run dev
```
